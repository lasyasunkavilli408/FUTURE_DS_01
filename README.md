# Business Sales Performance Analysis

## 📌 Overview
Analyzed business sales data to identify trends, top-performing products, and profit insights.

- Top products contribute major revenue

- dataset (Superstore.csv)
- analysis notebook (.ipynb)
- visualizations (images folder)

## 🚀 Outcome
Built a complete data analysis workflow including data cleaning, processing, and visualization.
